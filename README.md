# Welcome to My Backend Dropbox
***

## Task
Build a Dropbox-like serverless application with React, Amplify, Lambda, and S3, where the challenge is combining authentication, file storage, versioning, and cloud deployment in one working project.

## Description
This project solves the task with a React frontend and an AWS serverless backend using Amplify for authentication, Cognito for user accounts, and room for Lambda and S3 integration as the file upload and versioning features are completed.

## Installation
Install dependencies with `npm install`, initialize Amplify with `amplify init`, add authentication with `amplify add auth`, deploy backend resources with `amplify push`, and run the frontend with `npm run dev`.

## Usage
Create an account with an email address, confirm the verification code from Cognito, sign in through the Amplify login screen, and use the app dashboard to access the authenticated area while the remaining file features are being connected.
```
./my_project argument1 argument2
```
